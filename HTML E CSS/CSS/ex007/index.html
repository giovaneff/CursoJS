Height define a altura de uma caixa

Width defina a largura de uma caixa

Border é a linha que marca a caixa

Padding é o espaçamento dentro da caixa

Margin é o espaçamento depois da caixa

Existe dois tipos de box model que são box-level e inline-level

Box-level

Box-level é um modelo de caixa que realiza uma quebra de linha ou seja não da continuiadade ao elemento anterior, e também ocupa a linha inteira, um exemplo da mais utilizada é a DIV

Inline-Level

A inline-level não realiza a quebra de linha e a dimensão da caixa é o mesmo do elemento contido por padrão, um exemplo da mais utilizada é o SPAN
